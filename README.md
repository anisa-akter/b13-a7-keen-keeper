# KeenKeeper

KeenKeeper is a friendship management web app designed to help you stay connected with the people who matter most through quick check-ins, timeline tracking, and relationship insights.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- DaisyUI
- Recharts
- Lucide React Icons

## 3 Key Features

1. Personalized friend profiles with quick actions for call, text, and video check-ins.
2. Interaction timeline with search, filtering, and sorting to review relationship history.
3. Analytics dashboard that visualizes your interaction patterns with clear charts and summaries.
